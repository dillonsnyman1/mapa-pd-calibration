# Examples

Demo scripts that visualize the reference implementations' output. These are
not part of the validated reference implementations themselves (which have no
external dependencies) - they're for the project demo only.

Both scripts plot the unsmoothed (pooled-band step function) and smoothed
(log-odds interpolated) score-to-PD mappings produced by `run_pipeline` on the
shared [`../reference/fixtures`](../reference/fixtures) data, via a common
[`plotting.py`](plotting.py) helper - so the two charts are directly
comparable.

```sh
pip install -r requirements.txt
```

## `example_plot_python.py`

Calls the Python reference's `run_pipeline` directly.

```sh
python3 example_plot_python.py
```

Output: `output/python_calibration.png`.

## `example_plot_cpp.py`

Reads CSV output generated by the C++ reference. First build and run the
generator:

```sh
cd cpp
g++ -std=c++17 -I../../reference/cpp generate_output.cpp \
    ../../reference/cpp/mapa.cpp -o generate_output
./generate_output
cd ..
python3 example_plot_cpp.py
```

Output: `output/cpp_calibration.png`.
